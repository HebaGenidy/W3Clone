Not Sure Where To Begin?

HTML

The language for building web pages

HTML Example:

<!DOCTYPE html>
<html>
<head>
<title>HTML Tutorial</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

CSS

The language for styling web pages

CSS Example:

body {
    background-color: lightblue;
}

h1 {
    color: white;
    text-align: center;
}

p {
    font-family: verdana;
}

JavaScript

The language for styling web pages

JavaScript Example:

<button onclick="myFunction()">Click Me!</button>

<script>
    function myFunction() {
        let x = document.getElementById("demo");
        x.style.fontSize = "25px";
        x.style.color = "red";
    }
</script>

Python

A popular programming language for web and data

Python Example:

print("Hello, World!")

x = 5
y = 10
sum = x + y
print("Sum is:", sum)

SQL

A language for accessing and managing databases

SQL Example:

SELECT * FROM Customers
WHERE Country = 'Mexico';
